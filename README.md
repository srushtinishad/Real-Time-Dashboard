# 🌦️ City Wise Weather Analysis Dashboard

## 🚀 Project Overview

This project presents an **interactive Power BI dashboard** for analyzing real-time weather conditions across multiple cities. It provides insights into temperature variations, weather conditions, and atmospheric metrics to support comparative analysis between cities.

---

## 📌 Key Metrics

* **City Selected:** Mumbai
* **Temperature:** 29.99°C
* **Feels Like:** 34.08°C
* **Wind Speed:** 7.2
* **Max Temperature:** 29.99°C
* **Min Temperature:** 29.94°C
* **Country:** IN

---

## 📊 Dashboard Features

### 🔹 1. City Wise Weather Map

* Interactive map displaying weather conditions across:

  * Mumbai
  * Bangkok
  * Seoul
  * Tokyo
* Color-coded indicators for:

  * Clear
  * Clouds
  * Haze

---

### 🔹 2. Temperature Overview

* Displays key temperature metrics:

  * Actual temperature
  * Feels like temperature
  * Minimum and maximum temperature
* Enables quick understanding of real-time weather conditions

---

### 🔹 3. Feels Like vs Actual Temperature

* Comparative analysis between:

  * Actual temperature
  * Perceived (feels like) temperature
* Helps understand environmental impact such as humidity

---

### 🔹 4. Timestamp-Based Analysis

* Tracks temperature readings based on specific timestamps
* Allows monitoring of weather changes over time

---

### 🔹 5. Min vs Max Temperature Distribution

* Visual comparison of minimum and maximum temperature values
* Helps identify temperature variation range

---

### 🔹 6. Temperature by City

* Displays temperature distribution for selected cities
* Enables quick comparison across locations

---

### 🔹 7. Interactive Filters

* City selection:

  * Bangkok
  * Mumbai
  * Seoul
  * Tokyo
* Dynamic filtering updates all visuals instantly
